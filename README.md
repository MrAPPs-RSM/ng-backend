# Ng2Backend

## Clone & download

```bash
git clone https://github.com/MrAPPs-RSM/ng-backend.git
```

## Install

```bash
npm install
```

## Run locally

```bash
npm start
```

App will be served in: 

```bash
http://localhost:3000
```


## Deploy

```bash
npm run build:prod
```

All static files will be generated into /dist directory

For docs go to: [wiki](https://github.com/MrAPPs-RSM/ng-backend/wiki/)
